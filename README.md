# 营销策略

## 娱乐向（增量1）

主要是给普通大众使用（比如手机上的APP）

- 化妆
- 换头
- 表情识别
- .....

营销策略：免费，功能强大 ，注重用户隐私保护

目的：扩大产品影响力，提高知名度
